Quick Start

1.       Under Categories, click  and enter a name in the dialog that appears.

2.       Under Fields, click  and enter a name. You may add multiple fields for each category.

3.       Click  to add a new account record.

4.       Type in your entries (changes are saved automatically).

 

Terminology

Categories

Categories are containers in which accounts are stored. At least one category must be created before accounts can be added to the Shed.

Example categories: Work, Social Media, Banking.

 

Fields

Fields describe the type of data being stored. They are represented by columns in the Shed. Fields belong to categories and can vary for each category.

For example, accounts under a Social Media category may require fields such as URL, Username, and Password. Banking accounts may require fields such as Routing Number and Account Number.

 

Accounts

Accounts are represented by rows in the Shed. Each account belongs to a category.

 

Importing a .CSV File

1.       Under Categories, select the category you would like to import the .CSV file into.

2.       Click Tools -> CSV Importer

3.       Depending on your settings, a formatting guide may appear. Be sure your .CSV files format matches the specifications in the guide.

4.       Select the .CSV file you would to import in the dialog that appears.

5.       The CSV Import form should appear. Ensure that the desired category (selected in Step 1) is selected.

Updating

PassShed will automatically check for updates in the background each time the application is started. To manually check for updates, please visit the PassShed web page.

 

Troubleshooting

Its better for both of us if youre the one doing something wrong. Be sure to monitor status messages at the bottom of the window to troubleshoot unexpected behavior.
